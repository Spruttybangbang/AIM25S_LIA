# Scripts

Denna mapp innehåller alla Python-scripts för projektet, organiserade i underkataloger.

## Struktur

- **analysis/** - Analysskript för databas och datakvalitet
- **database_management/** - Hantering och underhåll av databaser
- **export/** - Export av data till olika format
- **scb/** - Integration med SCB (Statistiska Centralbyrån)

Se README i varje undermapp för detaljer.

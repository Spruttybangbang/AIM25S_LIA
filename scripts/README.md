# Scripts

Denna mapp innehåller alla Python-scripts för projektet, organiserade i underkataloger.

## Struktur

- **analysis/** - Analysskript för databas och datakvalitet
- **database_management/** - Hantering, underhåll och export av databaser
- **scb/** - Integration med SCB (Statistiska Centralbyrån)

Se README i varje undermapp för detaljer.

## Omfattande Guide

För en komplett guide till alla scripts, se [SCRIPTS_GUIDE.md](/SCRIPTS_GUIDE.md) i projektets root.

## Arkiverade Scripts

Scripts som inte längre används aktivt finns i `/archive/migrations/`. Se README där för detaljer.
